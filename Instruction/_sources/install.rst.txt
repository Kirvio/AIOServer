Как установить
===================================
 
Для установки(ручной) потребуется:
  
   Убедитесь, что у вас установлена нужная версия Python 3.9(python --version)

   Установите все дополнительные модули, что используется в этом проекте:

      - pip3 install aiosqlite;
      - pip3 install cryptography;
      - pip3 install xlsxwriter;
      - pip3 install tkcalendar;
      - pip3 install bcrypt;

Создайте новый каталог на диске C, назовите его Programm и переместите туда
все файлы из папки на сервере.

После всех вышеописанных действий, нужно будет запустить программу ClientForRecords.pyw

Для упрощенной установки нужно воспользоваться файлом install.bat(переместив все файлы в одну директорию)
