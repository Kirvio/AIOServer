Гайд для пользователей
===================================

Начало работы
-----------------------------------

- Запустите приложение с расширением `.pyw`
- После запуска появится окно для ввода логина и пароля:

.. image:: C:\\Users\\evandrushenko\\docks\\_templates\\Records.jpg
 
- Введите имя пользователя и пароль(для выдачи обращайтесь к администраторам)
- После входа в систему появится главное окно

.. image:: C:\\Users\\evandrushenko\\docks\\_templates\\Records1.jpg

Добавление записи
-----------------------------------

| Что-бы добавить запись:

| Заполнить все поля ввода корректно(без английских символов, длина предложения не должна превышать 100 символов)
| `Предварительно очистить поля для заполнения, можно нажав на кнопку` **Очистить поля ввода**.

| В поле **Выберите категорию**, выберите категорию заявки в выпадающем списке:

  - Телевидение
  - Интернет
  - Пакет

| В поле **Выберите дату**, нужно выбрать, на какое число вызван мастер
| Состояние заявки выбираете **Открыта**, иначе появится предупреждение
| Во все остальные поля вводите информацию в соответствии с описанием поля
 
| После того, как все поля будут корректно заполнены, можно:

 - Добавить запись нажав на кнопку **Добавить запись**
 - Выбрав в меню **Таблица** -> **Добавить запись**

Изменение записи
-----------------------------------

| Что-бы изменить запись:

 - Заполнить все поля(можно два раза нажать на нужную заявку в таблице)
 - Нажать на кнопку **Изменить запись**

Удаление записи
-----------------------------------

| Что-бы удалить запись:

 - Заполнить все поля(можно два раза нажать на нужную заявку в таблице)
 - Нажать на кнопку **Удалить запись**
 
 Поиск по таблице
 ----------------------------------
 
| Для поиска по таблице:

- Выберите пункт **Поиск** в верхнем меню
- Поиск по таблице осуществляется одним из трех способов:

  - По категории
  - По дате
  - По ФИО

- Для поиска по этим критериям, нужны заполненые поля категории, даты и ФИО.

Для администраторов
----------------------------------

| Для администраторов, помимо обычного функционала программы, существует окно для администрирования пользователей(в меню пункт Администрирование)

.. image:: C:\\Users\\evandrushenko\\docks\\_templates\\Records2.jpg

- Можно зарегистрировать пользователя, нажав на кнопку **Зарегистрировать** (все поля должны быть заполнены)
- Удалить пользователя по ID, заполнив поле ID(посмотреть ID можно в таблице) и нажав на кнопку **Удалить по ID**
