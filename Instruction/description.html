Описание программы¶

Эта программа используется для управления потоком заявок на вызов мастера от пользователей.
Программа клиент-серверная, что значит без сервера она работать не будет.
В качестве клиента используется GUI Tkinter, в качестве сервера asyncio tcp-server.
Описание функционала главного окна:
Слева находится поля для ввода с описанием, справа сама таблица с заявками.
В верхнем левом углу главного окна появляется ФИО текущего пользователя.
В верху окна есть меню, которое состоит из пунктов:

  • Меню

  • Поиск

  • Таблица

  • О программе

  • Выйти

  • Администрирование(для администраторов)

Меню:

  • Скрыть меню (что-бы скрыть поля для ввода)

  • Показать меню (вернуть все в обратное состояние)

  • Экспорт в Excel (экспорт данных из таблицы для отчета)

Поиск:

  • По категории (для поиска данных в таблице по выбранной категории)

  • По дате (для поиска по выбранной дате)

  • По ФИО (для поиска по ФИО абонента(ФИО должно быть введено полностью))

Таблица:

  • Все заявки/Обновить (для просмотра всех заявок из БД в таблице)

  • Добавить запись (выполняет ту же функцию, что и кнопка Добавить запись)

О программе (содержит ссылку на инструкцию по работе с программой)
Кнопки:

  • Добавить запись

  • Изменить запись

  • Удалить запись

  • Поиск по дате

  • Очистить поля ввода

Сама таблица содержит поля с информацией, соответствующей названию поля:

  • Дата выполнения заявки

  • ФИО

  • Адрес

  • Телефон

  • Причина

  • Услуга

  • Время выполнения

  • Для Мастера

  • Мастер

  • Состояние заявки

  • Категория

  • ФИО сотрудника

  • Дата регистрации
